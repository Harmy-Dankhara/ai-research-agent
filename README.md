# 🤖 AI Research Agent

An AI-powered Research Agent built using **LangChain** and **LangGraph** that generates structured research reports from user-defined topics.

The system supports persistent memory, streaming responses, and multi-turn conversational reasoning similar to ChatGPT.

---

## 🚀 Features

- 📚 Topic-based structured research generation
- 🧠 Persistent memory across sessions
- 🔄 Streaming LLM responses
- 💬 Multi-turn conversational reasoning
- 🗂 Chat history retention after system restart

---

## 🛠 Tech Stack

- Python
- LangChain
- LangGraph
- OpenAI API
- Gradio
- DuckDuckGo Search
- Tiktoken

---

## ⚙ Installation

1. Clone the repository:

```bash
git clone https://github.com/Harmy-Dankhara/ai-research-agent.git
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Add your OpenAI API key in a `.env` file:

```
OPENAI_API_KEY=your_api_key_here
```

4. Run the Python script:

```bash
python 1_final_research_maker.py
```
